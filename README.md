# 🛡️ MinorMint-AI-Shield
An AI-powered smart vault for teenagers using Google Gemini and Circle USDC on Polygon.

## 🚀 Overview
MinorMint is an intelligent financial agent that acts as a "Social Guardian" for minors. It evaluates spending requests using AI to ensure they align with safe and educational goals before authorizing transactions through Circle's programmable wallets.

## ✨ Key Features
* **AI Guardian:** Real-time analysis of purchase intent.
* **Smart Vaults:** Automated fund locking and security.
* **Blockchain Ready:** Powered by Circle USDC and Polygon.

## 📁 Project Structure
* `main.py`: The core engine of the agent.
* `guardian.py`: AI Safety and Social Reasoning logic.
* `integration.py`: Circle Wallet and Vault connectivity.
* `requirements.txt`: Project dependencies.
